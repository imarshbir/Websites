How to upload images for free:

  1. Upload your event images to a GitHub repository.
  2. Open the image on GitHub, click "Raw", and copy the URL.
  3. Paste that raw URL here (it starts with https://raw.githubusercontent.com/...).
  4. You can also use other hosting platforms like PostImages, Imgur, or Google Drive (direct links only).

🚀 Save & Publish Updates to Live Site

How to publish changes to Render / GitHub Pages:

  1. Make your edits above (Add or delete images). Changes are saved to your browser cache for instant previewing.
  2. Click the green "Export Data (data.json)" button below to download the compiled configuration file.
  3. Place this downloaded `data.json` file into your project's root folder (the folder containing `index.html`).
  4. Commit and push the `data.json` file to your GitHub repository (or upload it directly on GitHub.com).
  5. Once GitHub Pages or Render rebuilds (takes 1-2 minutes), the changes will immediately go live for every visitor in the world!
